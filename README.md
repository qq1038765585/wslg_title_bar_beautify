# 目前改动:
1. 使用按钮来切换light/dark模式, WSLG系统发行版是只读系统保存不了状态，系统重启后会恢复light模式
2. 使用Ctrl + 右键然后拖动调整窗口大小, 因为调整了边框边距，使用上下左右四个角很难调整
3. 使用Ctrl + F11 全屏

4. 对C语言不熟一通胡改可能会有BUG
